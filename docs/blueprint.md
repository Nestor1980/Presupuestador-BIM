# **App Name**: BIM Budgeteer

## Core Features:

- Basic Configuration: Allow users to configure basic settings such as value per hour, regional percentages, dollar exchange rate, surface area, and Level of Development (LOD).
- BIM Use Selection: Enable users to select BIM uses relevant to their project phase (Planning, Design, Construction, Operation) via a checklist, referencing the BIM Uses document from Penn State University.
- LOD Visualization: Visually represent the selected Level of Development (LOD) with graphical elements, highlighting the chosen LOD in the initial configuration screen.

## Style Guidelines:

- Primary color: Use a professional and calming blue (#3498db) for the header and main sections.
- Secondary color: Light grey (#f2f2f2) for backgrounds to provide a clean and neutral canvas.
- Accent: Use a vibrant green (#2ecc71) to highlight interactive elements such as checkboxes, sliders, and the final cost estimate, providing a clear visual cue for actionable items and key results.
- Divide the app into clear, distinct sections for Basic Settings, BIM Uses, BIM LOD, and Deliverables. Use a tabbed interface or vertical sections to organize content.
- Incorporate simple, intuitive icons to represent each BIM Use and LOD level, enhancing visual understanding and ease of navigation.
